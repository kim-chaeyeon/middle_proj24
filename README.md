## 🍳 팀원 소개
### 조장: 김채연

### 팀원: 어근혁

- UI
- PAGE  : 메인페이지,  로그인/로그아웃, 회원가입, 마이페이지, 유저페이지, 관리자 페이지

[메인 페이지]
- 광고 베너 슬라이드 구현
- 상단 로그인 상황에 따라 (로그인, 회원가입) , (로그아웃, 마이페이지), 관리자 로그인시 (관리자페이지, 로그아웃, 마이페이지) 출력 기능 구현
- 유저 검색 및 프로필 이동 기능 구현
- 중단 각 페이지 이동 기능 구현

[로그인/회원가입]
- 일반 회원 및 카카오, 네이버, 구글을 통한 소셜 로그인 구현
- 아이디 찾기, 이메일 인증을 통한 비밀번호 찾기 구현
- 회원가입 구현

[마이페이지/ 유저페이지]
- 마이페이지를 회원가입시 작성했던 개인 정보 및 작성한 글 확인가능, 프로필 수정, 회원탈퇴 구현
- 유저페이지를 통해 해당 유저가 작성한 모든 글 및 정보 확인 기능 구현

[관리자]
- 권한 인증 관리자 일때만 관리자 페이지 접근허용 기능구현
- 모든 회원 정보 확인 기능구현
- 권한이 관리자 일때 일반 회원 임의 탈퇴 기능구현
- 권한이 관리자 일때 모든 게시글에 대해 삭제 권한 발급 기능 구현

### 팀원: 박현철

- ui 페이지
  - 채팅개설방, 채팅방, 회원 차단, 신고

- 기능
[채팅]
    - 로그인한 회원만 이용할 수 있도록 권한 설정
    - 웹소켓, 스톰프 실시간 오픈형 채팅 구현
    - 사용자간 차단 및 신고
    - 채팅방 생성, 수정, 삭제 기능 구현

## 🍳 로그인
  ![login](https://github.com/user-attachments/assets/44df3c62-d8cb-4076-9e65-27a3fdec87f4)
- 회원가입을 통해 생성된 아이디와 비밀번호로 로그인을 수행합니다.
- 카카오, 네이버, 구글을 통한 소셜 로그인은 버튼을 눌러 각 플랫폼에 로그인하면 자동으로 계정 생성과 동시에 로그인을 수행합니다.
- 아이디 찾기와 비밀번호 찾기는 이메일을 통해 가능합니다.
-----------------------------------------------------------------
## 회원가입
![signup](https://github.com/user-attachments/assets/baec6685-04fa-4a84-b7da-39297f7d595f)
- 회원가입은 전화번호, 아이디, 닉네임, 비밀번호, 이메일,나이, 성별, 지역, 선호음식은 필수로 입력해야합니다.
- MBTI와 인스타는 필수가 아니 선택 옵션입니다.
- 프로필사진도 필수이고 파일선택을 클릭하여 프로필 사진을 선택할 수 있습니다.
- 회원가입을 클릭하면 이메일로 인증번호를 전송합니다.
- 이메일에서 받은 인증번호를 입력하면 회원가입이 완료됩니다.

  ![myPage](https://github.com/user-attachments/assets/00d01511-347f-45db-ae9f-4b43fc308131)

## 🍳 게시물
 ### 맛집 추천
  ![list](https://i.ibb.co/RN7HDgd/2024-06-25-210703.png)
- 사용자는 맛집 등록하기를 클릭하여 맛집을 등록할 수 있습니다.
- 사용자는 제목, 내용, 이미지를 포함하여 특정 게시물의 세부정보를 볼 수 있습니다. 
- 사용자는 상세 페이지에서 수정 및 삭제가 가능합니다.
 ---------------------------------------------------------
 ### 밥 친구 구함
  ![list](https://i.ibb.co/ccV9KK3/2024-06-25-211229.png)
- 밥친구 페이지에서는 제목, 내용, 인원, 주소, 시간 및 날짜를 입력해 등록할 수 있습니다.
- 주소 클릭 시 세부 내용을 확인 할 수 있습니다.
- 세부 내용에서는 같이 먹기를 눌러 인원이 얼마나 있는지 확인 할 수 있습니다.
- 사용자는 상세 페이지에서 수정 및 삭제가 가능합니다.
-------------------------------------------------------------
 ### 밥먹다D.S
  ![list](https://i.ibb.co/S7rQtck/2024-06-25-212446.png)
- Daily Story에서는 제목, 내용, 사진으로 입력해 등록할 수 있습니다.
- 로그인한 회원은 모두 볼 수 있습니다.
- 여러 회원들이 댓글을 달 수 있으며 나와 사람들의 일상적인 스토리를 볼 수 있습니다.
- 사용자는 상세 페이지에서 수정 및 삭제가 가능합니다.
-------------------------------------------------------


### 🍳 채팅
  ![채팅](https://i.ibb.co/W5fkD53/2024-06-26-151534.png)
- 오픈형 채팅방을 생성하고 여러 사람들이 참여할 수 있습니다.
- 채팅방 삭제는 채팅방을 만든 회원만 삭제할 수 있도록 했습니다.
- 개설된 채팅방을 클릭하면 채팅방으로 이동합니다.
----------------------------------------------------------
  ![채팅](https://i.ibb.co/Tmvc6D3/2024-06-26-151644.png)
- 채팅방에 참여해 실시간으로 상대방과 채팅할 수 있습니다.
- 이미지 파일도 첨부하여 보낼 수 있도록 구현하였습니다.
- 채팅방에 참여한 인원의 닉네임을 확인할 수 있습니다.
- 부적절한 언행이나 욕설을 할 경우 닉네임을 클릭하면 클릭한 회원의 정보 페이지에 들어가 차단이나 신고를 할 수 있도록 했습니다.
